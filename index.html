Nimble Health

Katie Yawney, Jake Durell and I created the software for Nimble Health, a transportation service for health care providers that ensures patients in rural areas have the transportation they need to reach their appointments. Using Node, Express, Google Maps, Bootstrap and MongoDB we developed separate interfaces for the two main user types - administrators and drivers. The driver interface is optimized for mobile devices.
